# 📦 BetbetMiro Extension

Koleksi multi-extension provider untuk Cloudstream berbasis CS3 plugin system.

Proyek ini menyediakan berbagai source streaming dalam satu repository dengan sistem build otomatis menggunakan GitHub Actions.

---

## ⚙️ Fitur

- 🔌 Multi-extension dalam satu repository  
- 📦 Output build `.cs3` otomatis  
- ⚡ CI/CD via GitHub Actions  
- 🧩 Plugin registry (`plugins.json`) auto-generated  
- 🔄 Update extension berbasis versioning  
- 🛠 Struktur modular per provider
